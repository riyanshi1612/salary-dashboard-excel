 📊 Employee Salary Analytics Dashboard

Excel • PivotTables • PivotCharts • Data Visualization

## Overview

An interactive Excel dashboard analyzing 10,000+ real-world data science job postings across multiple countries, roles, and job platforms. Built to surface salary insights by job title, country, and employment type.

## Features

✅ Interactive salary calculator with dynamic dropdowns (Job Title, Country, Type)

✅ PivotTable showing average salary by job title with slicer for job type filtering

✅ PivotChart — horizontal bar chart sorted by salary

✅ World map visualization of salary distribution by country

✅ Salary by employment type chart (Full-time, Contractor, Part-time etc.)

✅ KPI cards — Median Salary ($90K), Top Hiring Platform (Indeed), Job Count (6,480)

## Skills Used

MS Excel, PivotTables, PivotCharts, Slicers, Data Validation, MEDIAN/IF formulas, Dashboard Design, Data Cleaning

## Dataset
Real-world data science job postings dataset containing job title, salary, country, employment type, job platform, and required skills.

## Key Insights

📌 Senior Data Scientists earn the highest average salary (~$154K/year)

📌 Indeed is the top hiring platform for data science roles

📌 Full-time roles offer the highest median salary ($90K) vs other employment types

📌 United States dominates data science job postings globally
